LOYALTY PROGRAM - Instructions to run

How to compile:

1. Open terminal or command prompt in that folder.
2. Run the following command to compile the code:

   g++ -std=c++17 main.cpp User.cpp EcomLoyaltySystem.cpp LoyaltyProgram.cpp -o EcomLoyalty

   This will generate an executable file named `loyalty_program`.


How to run:

After compilation, run the following command:

   ./EcomLoyalty
